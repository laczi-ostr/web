What is python?

What is Python that is good question. Of course we don't speak about a snake in grass. Python is elegant open source programming language that delivers both the power and general applicability of traditional compiled languages. It is optimized for software quality, developer productivity, program portability, and component integration. Python is used by at least hundreds of thousands of developers around the world in areas such as Internet scripting, systems programming, user interfaces, product customization, numeric programming, and more. It is generally considered to be among the top four or five most widely-used programming languages in the world today.
